Diccionario de Domótica

1. Domótica

La domótica se refiere al conjunto de sistemas y tecnologías orientadas a automatizar las funciones de una vivienda o edificio.

2. IoT (Internet of Things)

El Internet de las Cosas se refiere a la red de dispositivos físicos, vehículos, electrodomésticos y otros objetos que están integrados con tecnología para conectar e intercambiar datos a través de Internet.

3. Sensores

Dispositivos que detectan y responden a estímulos físicos o ambientales y envían la información captada a otros dispositivos o sistemas.

4. Actuadores

Componentes de un sistema domótico que ejecutan acciones físicas, como encender o apagar luces, abrir o cerrar puertas, etc.

5. Automatización

Capacidad de un sistema domótico para realizar tareas automáticamente según programación o condiciones del entorno.

6. Gateway

Dispositivo que actúa como puente entre diferentes protocolos de comunicación utilizados en sistemas domóticos.

7. Protocolo de Comunicación

Estándar utilizado para que los dispositivos domóticos se comuniquen entre sí, como Zigbee, Z-Wave, etc.

8. Hub

Dispositivo central en un sistema domótico que controla la comunicación entre los diferentes dispositivos.

9. Interface de Usuario (UI)

Interfaz gráfica a través de la cual los usuarios interactúan y controlan los dispositivos domóticos.

10. Red Mesh

Red de malla que permite una comunicación más robusta y eficiente entre los dispositivos domóticos distribuidos por toda la casa.

11. Domótica Inalámbrica

Sistemas domóticos que utilizan tecnologías inalámbricas como Bluetooth, Wi-Fi, Zigbee, etc., para la comunicación entre dispositivos.

12. Sistema de Seguridad Domótica

Conjunto de dispositivos y tecnologías que protegen la vivienda mediante la detección y alerta de intrusiones, incendios, fugas de gas, etc.

13. Smart Speaker

Altavoz inteligente que permite controlar dispositivos domóticos mediante comandos de voz.

14. Geolocalización

Tecnología que utiliza la ubicación física de los dispositivos móviles para activar o desactivar dispositivos domóticos según la ubicación del usuario.

15. Sistema HVAC

Sistema de calefacción, ventilación y aire acondicionado que puede ser controlado de manera inteligente mediante sistemas domóticos.

16. Escenas Domóticas

Configuraciones predefinidas que permiten activar múltiples dispositivos domóticos con un solo comando o acción.

17. Energía Solar Domótica

Sistemas domóticos que gestionan la generación, almacenamiento y consumo de energía solar en una vivienda.

18. Wearables en Domótica

Dispositivos tecnológicos que se llevan puestos y que están integrados en sistemas domóticos, como relojes inteligentes, pulseras de actividad, etc.

19. Domótica DIY (Hazlo tú mismo)

Sistemas domóticos que pueden ser instalados y configurados por el usuario sin necesidad de contratistas o instaladores profesionales.

20. Integración de Dispositivos

Capacidad de diferentes marcas y tipos de dispositivos domóticos para trabajar juntos en un mismo sistema, proporcionando una experiencia de usuario integrada.
